## Why Open Crypto Checkout?

<CardGrid stagger>
  <Card title="Headless & Flexible" icon="puzzle">
    No bundled UI. Ship your own checkout frontend and call the API directly. Full control over the user experience.
  </Card>
  <Card title="Multi-Chain" icon="seti:puppet">
    Accept USDT, USDC, and DAI across Ethereum, Polygon, Base, Arbitrum One, Optimism, Linea, Avalanche C-Chain, and Tron.
  </Card>
  <Card title="Self-Hosted" icon="seti:bicep">
    Run on your own infrastructure. A single Docker container plus a PostgreSQL database is all you need.
  </Card>
  <Card title="Secure by Design" icon="seti:lock">
    HMAC-signed requests, Argon2-hashed admin secrets, and origin-verified checkout URLs keep your payments safe.
  </Card>
</CardGrid>
